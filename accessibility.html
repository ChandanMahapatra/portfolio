Overview

The Integrated Assistive System was my final year undergraduation project. I worked with teammates on developing an integrated assistive technology using Morse Code. The project was developed to be socially relevant by assisting people suffering from Motor Neuron Disease. We designed the prototype with the primary goal being that it would be easy to use and affordable.

Process


We decided to use Morse Code as it is essentially a binary form of communication and we could use strings of dot’s and dashes to create sentences. Morse Code can also be used for many languages apart from English.

The focus in this project was primarily on the user interface aspect. The goal was to make it simple to use while maintaining its multiple applications. We did this by using four push button switches that would be used as the primary input.

Push button switches were used rather than touch screen as touch screen devices are too sensitive to be used by the patients. As the patient’s condition deteriorates, it becomes more difficult for them to operate these touch screen devices.

Block Diagram of the Transmitter

The four simple push button switches represented the dot, dash, next character and clear as the input system to the Arduino, where the input is provided by the patient or user by pressing the keys in sequence to represent a letter in Morse code.

The Arduino kit is interfaced with a push button switch and the Arduino translates the message encoded in Morse code into English. The message is also sent wirelessly to the receiver block using Zigbee Modules that communicate serially.

The receiver block displays the message on a LCD screen. Ten seconds after the last message character is typed, a buzzer beeps six times to indicate the end of a message.

When a numeric character assigned to either turn on or turns off an AC powered appliance is entered using the push buttons, the relay switch connected to the device completes the switching circuit. The change in status of the device is indicated on the LCD of the Receiver side.
